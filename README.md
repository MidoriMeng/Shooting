# Shooting  
大三下半学期之后的小学期的作业，和一位美术合作完成的第一人称射击游戏，有基于状态机的简单AI。
本次最大的收获是体验合作&制作美术素材，包括设计并制作地图、人物模型（使用magicaVoxel和blender），制作人物骨骼并绑定（使用blender），导入unity中添加免费动画（来自UnityChan）
